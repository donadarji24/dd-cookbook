# dd-cookbook
A modern recipe website featuring a photo gallery and detailed cooking instructions, designed to showcase dishes visually and share easy-to-follow recipes.

dd-cookbook is a web project that combines a visual food gallery with a collection of recipes. The website allows users to explore dishes through high-quality photos and view detailed recipes including ingredients, preparation steps, and cooking tips.

The goal of this project is to create an engaging cooking platform where food photography and recipe content are presented in a clean, user-friendly interface. It can be used as a personal cookbook, a recipe sharing platform, or a portfolio for food photography and web development.
